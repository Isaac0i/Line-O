Cada pedacinho de conteúdo que você vê (um título, um parágrafo, uma imagem, um link) é um elemento HTML. Eles não são apenas texto; são marcadores que dão significado e estrutura ao que o navegador vai exibir. Olha Alguns Exemplos !

Elementos Linha Existem 3 tipos de elementos no HTML, tratando-se de seu comportamento de renderização, os chamados:
Elementos Bloco (Block-level)
Elementos Linha (Inline)
Elementos Linha (Inline-block)
São elementos que se comportam como contêiners, ocupam 100% do espaço disponível e sempre começam em uma nova linha.
São elementos que continuam na mesma linha,ocupam apenas o espaço disponível do conteúdo, comportando-se de uma forma linear.
São elementos intermediários que continuam na mesma linha e aceitam alterações em relação a altura e largura.

Semântica !
A semântica serve para representar o significado conteúdo, principalmente aos navegadores, é como dizer aos navegadores e os mecânismos de busca o que cada elemento representa, olha alguns exemplos :
<nav> = Elemento que representa o menu de navegação da página.
<main> = Elemento que representa o conteúdo principal e único da página.
<article> = Elemento que representa um conteúdo independente, como um post de blog.
<section> = Elemento que representa uma seção temática da página.
<footer> = Elemento que representa o rodapé da página.
<div> = Elemento que representa uma caixa genérica de bloco usada para agrupar elementos quando nenhuma tag semântica se encaixa.
<form> = Elemento que representa os formulários, feito para coletar os dados do usuário.
<button> = Elemento interativo usado para ações e eventos na página.

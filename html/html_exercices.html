HTML

Exerícios

1. Qual é a principal diferença entre um elemento HTML semântico e e um elemento genérico ?

A. Elementos Semânticos são apenas para estilização, enquanto gecêricos definem a estrutura. B. Elementos semânticos não podem ser estilizados com CSS, mas genéricos sim. C. Elementos semânticos descrevem o significado do conteúdo, enquanto genéricos são contêineres sem significado específico. D. Elementos genéricos sempre começam em uma nova linha, e semânticos não.

2. Qual é a característica principal de um elemento HTML de bloco ?

A. Ele ocupa apenas o espaço do seu conteúdo e não quebra a linha. B. Ele pode ser estilizado com CSS, mas não aceita altura ou largura. C. Ele é usado principalmente para formatar texto dentro de um parágrafo. D. Elementos genéricos sempre começam em uma nova linha, e semânticos não.

3. Um dos maiores benefícios de usar HTML semântico é a melhoria na acessibilidade. Por quê?

A. Porque ele faz o site carregar mais rápido para todos os usuários. B. Porque ele automaticamente adiciona cores e estilos para contraste. C. Porque ele evita a necessidade de usar JavaScript para interatividade. D. Porque ele permite que leitores de tela entendam a estrutura e o propósito do conteúdo para usuários com deficiência visual.

4. Quando é o momento mais apropriado para usar um elemento “ < div > ”?

A. Quando você precisa agrupar elementos puramente para estilização ou manipulação JavaScript, e não há uma tag semântica mais específica. B. Para criar um link para outra página. C. Sempre que o conteúdo for um título principal “ < h1 > ”. D. Para marcar um texto de forte importância dentro de um parágrafo.

5. Qual o comportamento de um elemento com “display: inline-block”?

A. Ele faz o texto ficar em negrito e itálico ao mesmo tempo. B. Ele só pode conter outros elementos de bloco. C. Ele flui como um elemento em linha, mas aceita propriedades de largura, altura, margem e preenchimento completos. D. Ele sempre começa em uma nova linha e ignora propriedades de largura e altura.

6. Qual elemento HTML semântico é mais apropriado para envolver um bloco de informações de contato do autor de um artigo específico?

A. <div> B. <footer> C. <address> D. <article>

7. Um dos benefícios da semântica HTML é a melhoria na manutenibilidade do código. O que isso significa?

A. O código HTML conserta erros sozinho. B. O código se torna mais fácil de entender e modificar por outros desenvolvedores (ou por você no futuro). C. O site nunca precisará de atualizações. D. O site funciona mais rápido em dispositivos antigos.

8. Qual é o propósito do elemento “ < footer > ”?

A. Para agrupar o conteúdo principal e exclusivo de uma página. B. Para criar um rodapé fixo na parte inferior da tela, que não rola. C. Para definir o cabeçalho principal de uma seção ou do documento. D. Para conter informações de autoria, direitos autorais, navegação secundária ou informações de contato de uma seção ou do documento.
